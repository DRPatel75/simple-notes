# Simple Notes App

A clean and simple Notes application built using **React + Vite**.  
This project demonstrates reusable React components, basic UI structure, and modern frontend development setup using Vite.

---

## 🚀 Features

- Create a simple notes UI
- Reusable React components
- Fast development with Vite
- Organized project structure
- Responsive design support
- Easy to customize
